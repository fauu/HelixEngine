## Technology

* Java
* [libGDX] framework (engine core)
* [artemis-odb](https://github.com/junkdog/artemis-odb) Entity-Component-System framework (engine core)
* [VisUI](https://github.com/kotcrab/vis-ui) UI library (editor UI)
